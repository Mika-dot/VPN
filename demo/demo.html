Bitmessage

Bitmessage is an open source, decentralized P2P email messaging system that allows a Bitmessage user to send encrypted messages to other system users as an alternative to email. Bitmessage is built on an architecture similar to the Bitcoin network, but adapted for the task of sending messages rather than monetary transactions. The network has exploded in popularity after Edward Snowden's disclosure of a secret surveillance system for Internet users, PRISM.

Freenet

Freenet is a decentralized and completely anonymous peer-to-peer network that runs on top of the Internet, includes a large number of peer-to-peer computers and allows you to publish any material without the ability to trace the author. Data confidentiality is guaranteed by strong cryptography: in order to receive a file, a request must provide the key associated with it. The role of such a key is played by the file hash code or DSA key, which also forms an integrity check mechanism. Freenet is now starting to use the Onion Routing networking principle.

SKad

Perfect Dark is a client for the anonymous file-sharing network SKad (OpenKAD) - a modification of the Kademlia protocol - developed in Japan to replace the previous Winny and Share. Similar in structure to Freenet, but uses DHT with a large distribution. The data is stored in encrypted blocks and transmitted separately from the keys. For encryption, RSA and AES algorithms are used, and the keys are cached to speed up file sharing. ECDSA is used to digitally sign files.

RetroShare

RetroShare is a free serverless program for exchanging emails, instant messages and files over an encrypted F2F (and optionally P2P) network built on GPG. Refers to the so-called. Darknet (overlay network), as peers can exchange certificates and IP addresses with their friends. Uses Turtle F2F for file sharing and DHT Kademlia for searching.

WASTE

WASTE is a file sharing program for use within trusted user groups. Includes IM, chat and online member database. Supports distribution of files both on individual computers with authentication support and throughout the network. All connections within the network are encrypted with the RSA algorithm, which ensures free and secure file sharing without the risk of eavesdropping.
